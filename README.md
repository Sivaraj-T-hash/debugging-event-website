# QR-Based Debugging Relay Website 🚀

A robust web application designed to facilitate a QR-code-based debugging competition. This platform allows event organizers to host a relay-style challenge where participants scan physical QR codes to unlock specific code-debugging tasks in real-time.

---

## 🛠 Tech Stack
* **Backend:** Python (Flask)
* **Database:** FireBase
* **Frontend:** HTML5, CSS3, JavaScript (AJAX for submissions)
* **Containerization:** Docker
* **Deployment:** LocalHost

## 🚀 Key Features
* **Dynamic QR Integration:** Each QR code maps to a specific challenge ID via URL parameters.
* **Live Debugging Interface:** Integrated code editor view for participants to fix snippets.
* **Automated Validation:** Instant feedback on code submissions.
* **Leaderboard:** Real-time tracking of team progress and scores.
