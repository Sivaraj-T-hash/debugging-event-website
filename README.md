# QR-Based Debugging Relay Website 🚀

A web application designed to facilitate a QR-code-based debugging event. Participants scan codes to access debugging challenges in a relay format.

## 🛠 Tech Stack
- **Backend:** Python (Flask)
- **Database:** MongoDB
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Docker, Render

## 🚀 Features
- QR Code integration for seamless challenge access.
- Real-time debugging interface.
- Automated score management.

## ⚙️ Setup
1. Clone the repo: `git clone https://github.com/Sivaraj-T-hash/debugging-event-website.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
